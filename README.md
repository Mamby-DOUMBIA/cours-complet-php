# 📘 Cours PHP Complet — Du Débutant à l'Expert

> **Formation professionnelle complète en PHP** — Rédigée comme un manuel universitaire et une documentation technique d'ingénieur backend.

---

## 🎯 Objectifs du cours

À l'issue de cette formation, vous serez capable de :

- ✅ Comprendre les fondamentaux du langage PHP et son rôle dans le Web
- ✅ Développer des applications Web dynamiques côté serveur
- ✅ Maîtriser la programmation orientée objet (POO) en PHP
- ✅ Construire des applications backend professionnelles et sécurisées
- ✅ Interagir avec des bases de données via PDO et MySQLi
- ✅ Comprendre et utiliser l'écosystème PHP (Composer, Laravel, Symfony)
- ✅ Écrire du code propre, sécurisé et maintenable

## 🚀 Comment utiliser ce cours

### Étape 1 — Préparez votre environnement

Avant de commencer, installez un environnement de développement local :

- 🔗 [XAMPP (Windows/Linux/Mac)](https://www.apachefriends.org/fr/index.html)
- 🔗 [WAMP (Windows)](https://www.wampserver.com/)
- 🔗 [MAMP (Mac)](https://www.mamp.info/en/)

> Voir le module **02-installation** pour un guide pas à pas complet.

### Étape 2 — Suivez les modules dans l'ordre

Chaque module contient :

1. Un fichier `README.md` — explication théorique détaillée
2. Des fichiers `.php` — exemples de code commentés
3. Un fichier `exercices.md` — exercices pratiques
4. Un fichier `corriges.md` — corrigés détaillés

### Étape 3 — Réalisez les projets pratiques

Une fois les modules terminés, construisez les projets dans `/projets/`.
